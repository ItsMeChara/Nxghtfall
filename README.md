SSL Certificate Untrust
u/Chara
Medium Risk
Easy Setup
Mar 1, 2026
Securly requires a specific SSL Certificate to inspect encrypted HTTPS traffic. Deleting or untrusting this certificate "blinds" the filter.

Step 1: Go to chrome://settings -> Privacy and security -> Security -> Manage certificates.
Step 2: Under the Authorities tab, find "Securly" or "Amazon/SecureTrust CA" entries used by the district.
Step 3: Click the three dots and select Delete or Untrust.
Result: Most filtered sites will now return an "SSL Error" instead of a block page. You can often bypass this error by typing thisisunsafe directly on the error page (no text box needed) to force the site to load.
